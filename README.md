# pwa-demo

## Project setup
```
npm install
```
### run the db.json, it works as JSON Server
```
json-server --watch db.json
```

### Compiles and hot-reloads for development
```
npm run serve
```

### Compiles and minifies for production
```
npm run build
```

### Lints and fixes files
```
npm run lint
```
